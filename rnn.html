Intuition: How Do RNNs Work?

It is as simple of a neural network can get. At every time instance $t$, it reades one token from input text, say $x_t$ and produces an output $y_t$ and a hidden state $h_t$.

Mathematical Definition:

$$h_t = f(W_{hh}h_{t-1} + W_{xh}x_t + b_h)$$ $$y_t = W_{hy}h_t + b_y$$
