Linear Regression

Let's start our machine learning journey from this. This is the stepping stone or the early ideas of Machine learning. Simple as it sounds and ancestor of mathemtical foundations in ML.

In Linear Regression, a dependent variable $y$ is associatetd with set of independent variables $(x_1, x_2, ..., x_d)$ via a linear predictor function.

For example, let's take a popular housing price problem. So, we have a bunch of data points of house prices. Say, we know different properties of houses (In ML term, we call them features or attributes). The features are area, number of bedrooms, kitchen size, garden, is parking available, location of house and so on. And we know the prices as well. So, typically the problem is given so many data points of house features and price, can we create a predictor function which can predict price for a new house given all the necesary features.
If we try to fit a linear function through the features, it will be a problem of Linear Regression.

Intuition: Foundation of Linear Regression

Let's define each term carefully. We assume to start with the housing price problem only. So, to make each prediction say we know $d$ different things about the house. Or in other words, our number of features are $d$.
$$X = \begin{bmatrix} x_0 \\ x_1 \\ \vdots \\ x_{d-1} \end{bmatrix}_{d \times 1}$$
So, our each independent variable matrix is like $X$. To compare with our example problem, $x_0$, $x_1$ are here bedroom size, area and so on. Now, we are trying to fit a linear function through the features. So, our predictor function is,
$$\begin{align*} y = f_w(x) + \epsilon \end{align*}$$
Here $f_w(x) = w_0 + w_1x_1 + w_2x_2 + ... + w_dx_d$. $w_i$ is the weight associated with $i$-th feature (Also known as parameter). $\epsilon$ is the irreducible error term (we will go into this in details later). So, in matrix form $W$ is
$$W = \begin{bmatrix} w_0 \\ w_1 \\ \vdots \\ w_{d-1} \end{bmatrix}_{d \times 1}$$

Pricing Again?

In the following example, we are trying to predict Rikshan price per distance travelled. The $x$ values are distance travelled (only one feature) and dependetent predicted variables are $y$. We plot the data points and try to predict the ideal linear function to fit that data. Now for example, we assume the ideal function is,
$$\begin{align*} y = w_0 + w_1x \end{align*}$$
Now, if we look at the graph, the fit looks more or less correct. We don't if it's the best fit mathematically yet. Speaking in mathematics, we can think there are infinitely many combinations of $(w_0, w_1)$, which leads to inifinitely many possible straight lines to fit. So, to find the mathematically most accurate one, we need an objective function to reduce the cost of fit. Let's look into that in next part.

But to suffice current goal of this exmaple, it looks a good fit and for a complete new data point of distance we need to travel, we can find the $y$ value corresponding to that $x$ and assume that's closest possible prediction to the actual price we have to pay.

More Maths? Ughhh!

A linear prediction model in the ways we define, means any non-linear transformation of the independent variable are also permitted.

So, $y = w_0 + w_1x + w_2x^2 + w_3 + x^3 + ... + w_kx^k$ is still linear predictor. To understand better please go through Linear & Non-Linear Transformations (ref 1)

Irreducible Error

If two variables are probabilistically related, then for a fixed value $x$, there is uncertainty in the value of the second variable.

We assume the uncertainty as,

$$ \begin{align*} y = w_0 + w_1x + \epsilon \end{align*} $$
$\epsilon$ is a random variable representing irreducible error. Two variables $(y, x)$ are related linearly 'on average' if fr fixed $x$ the actual value of $y$ differs from its expected value by a random amount (random error).
$$E(\epsilon) = 0$$ $$V(\epsilon) = \sigma^2$$

The errors are distributed about the ideal regression line. Hence, mean is considered to be $0$.

$$ \begin{align*} E(\epsilon) = \sum_{i} e_i = 0 \end{align*} $$ $$ \begin{align*} V(\epsilon) = \sum_{i} e_i^2 - \frac{(\sum_{i} e_i)^2}{n} = \sigma^2 \end{align*} $$

Remember, in Linear Regression we assume that the irreducible noise comes from a Gaussion Distribution.
If we think of an entire population of $(x, y)$ pairs then $\mu_{y/x^*}$ is the mean of all $y$ values for which $x=x^*$ and $\sigma_{y/x^*}$ is a measure of how much these values of $y$ spread out about the mean value.
How do we interpret $\sigma_{y/x^*}$?

Homoscedasticity

We assume the variance of the distribution of $y$ values to be the same at each different value of fixed $x$ (Homogeneity of variance assumption). In other words, those residuals are not dependent on $x$ values. They come from an independent Gaussian distribution with a constant variance $\sigma^2$.

Error Function / Loss Function / Cost Function

We have already seen that to find out the most suitable fit, we need a cost function to optimise. There are different ways to define a cost function.
The most common cost function is Least Squares Estimate.
Mathematically, we define MSE as folows. Let's assume we have a dataset $D$ with $n$ data points.

$L = \sum_{(x_i,y_i \in D)} (y_i - f_w(x_i))^2$

We use this the most because it's easy to differentiate and optimise. Let's see how in later parts.

Next one is Least Absolute Deviation Estimate.

$L = \sum_{(x_i,y_i \in D)} |y_i - f_w(x_i)|$

Mean Squared Logarithmic Error.

$L = \sum_{(x_i,y_i \in D)} (\log(y_i) - \log(f_w(x_i)))^2$



Some Basic Notations

$D = {(x_1,y_1), ..., (x_n,y_n)}$ -> Dataset
$n$ = number of training examples
$x_j$ = input or independent variables
$y_j$ = output or dependent variables or target variables
$(x_j, y_j)$ = specific examples ($j^{th}$ training example)

$x_j = \begin{bmatrix} x_j^1 \\ x_j^2 \\ \vdots \\ x_j^d \end{bmatrix}_{d \times 1}$ $X = \begin{bmatrix} x_1^T \\ x_2^T \\ \vdots \\ x_n^T \end{bmatrix}_{n \times d}$ $Y = \begin{bmatrix} y_1 \\ y_2 \\ \vdots \\ y_n \end{bmatrix}_{n \times 1}$


General Regression Problem

Determine a function $f^*$ such that $f^*(x)$ is the best predictor for y, w.r.t. D.
$$f^* = \arg\min_{f \in \mathcal{F}} E(f, D)$$
Here, $F$ denotes the class of functions over which error minimization is performed. $D$ is set of data points.

Parameterized Regression Problem

Need to determine parameters $w$ for the function $f(x, w)$ which minimizes our error function $E(f(w, x), D)$.
$$w^* = \arg\min_{w} E(f(w, x), D)$$ $$w^* = \arg\min_{w} \sum_{n} (y_i - w^Tx_i)^2$$
Now, for current scenario, we will begin with the most widely used loss function.

Closed Form Solution for 1D Data

- With the statement that Least Square Function is a Convex Function (ref) we can move ahead.
- Our goal is to find the minimal cost of our cost function.
- Which is the optima of this function.
This is a typical convex function of cost in linear regression. And our goal is to find that global minima as our ideal solution.
$$\begin{align*} w_0^*, w_1^* = \arg\min_{w_0, w_1} \sum_{n} (y_i - f(x_i))^2 \\ w_0^*, w_1^* = \arg\min_{w_0, w_1} \sum_{n} (y_i - w_0 - w_1x_i)^2 \end{align*}$$
So, we'll find $w_0^*$, $w_1^*$, when $$ \frac{\partial}{\partial w_0} E = 0$$ $$ \frac{\partial}{\partial w_1} E = 0 $$ Let's solve for it.

Case 1:

$$ \begin{align*} \frac{\partial}{\partial w_0} E(w_0, w_1, D) = 0 \\ or, \frac{\partial}{\partial w_0} \sum_{i=1}^{n} (y_i - w_0 - w_1x_i)^2 = 0\\ or, -2\sum_{i=1}^{n} (y_i - w_0 - w_1x_i) = 0\\ or, nw_0 + w_1\sum_{i} x_i = \sum_{i} y_i \quad \cdots (A) \end{align*} $$

Case 2:

$$ \begin{align*} \frac{\partial}{\partial w_1} \sum_{i=1}^{n} (y_i - w_0 - w_1x_i)^2 = 0\\ or, -2\sum_{i=1}^{n} (y_i - w_0 - w_1x_i)x_i = 0\\ or, w_0\sum_{i} x_i + w_1\sum_{i} x_i^2 = \sum_{i} x_iy_i \quad \cdots (B) \end{align*} $$
So from $A$, we can find,
$$ \begin{align*} w_0^* = \frac{\sum y_i - w_1^*\sum x_i}{n} = \bar{Y} - w_1^*\bar{X} \quad \cdots (C) \end{align*} $$
Substituting $w_0^*$ in $B$, we can find,
$$ \begin{align*} w_1^* = \frac{\sum_{i} x_iy_i - w_0^*\sum_{i} x_i}{\sum_{i} x_i^2}\\ w_1^* = \frac{\frac{\sum_{i}x_iy_i}{n} - \bar{X}\bar{Y}}{\frac{\sum_{i}x_i^2}{n} - \bar{X}^2} \quad \cdots (D) \end{align*} $$
Moving further, In matrix form,
$$\begin{align*} E^* = \arg\min_{w} \sum_{i} (y_i - w^Tx_i)^2\\ or, \nabla_{w}E =\sum_{i} -2x_i (y_i - w^Tx_i) = 0\\ or, -2\sum_{i}x_iy_i + 2\sum_{i}x_i^2w^T = 0\\ or, \nabla_wE = -2X^TY + 2X^TXw = 0\\ or, X^TXw = X^TY\\ or, \mathbf{\color{#2a51fa}w^* = (X^TX)^{-1}X^TY} \quad \cdots (E) \end{align*}$$
$X^TX$ is guranteed to be invertible if $X$ is a column vector.

Here, $\nabla_{w}E = \begin{bmatrix} \frac{\partial E}{\partial w_0} \\ \frac{\partial E}{\partial w_1} \\ \vdots \\ \frac{\partial E}{\partial w_{d-1}} \end{bmatrix}_{d \times 1}$
To understand better about matrix operations about differentiation, please read up.

Let's Revisit Least Squares

Let's try to re-do the closed form from a matrix perspective to get a better view of dimensionality and data.

Given a training data, we design matrix $X$ as an $(n \times d)$ matrix. And $Y$ is the $n$ dimensional target value vector.

$X = \begin{bmatrix} \cdots (x_1)^T \cdots \\ \cdots (x_2)^T \cdots \\ \vdots \\ \cdots (x_n)^T \cdots \end{bmatrix}_{n \times d}$ $Y = \begin{bmatrix} y_1 \\ y_2 \\ \vdots \\ y_n \end{bmatrix}_{n \times 1}$

$w$ is the weight vector along with bias factor. And $X$ has the unit term to get bias.

$E(x_i) = (x_i)^T w$

$E(X) = Xw$

$XW-\bar{Y} = \begin{bmatrix} (x_1)^T w - y_1 \\ (x_2)^T w - y_2 \\ \vdots \\ (x_n)^T w - y_n \end{bmatrix}_{n \times 1}$

We know that for a vector $Z$, $Z^TZ = \sum_{i} z_i^2$.

$$ \begin{align*} \frac{1}{2}(XW-\bar{Y})^T(XW-\bar{Y}) = L(w)\\ = \frac{1}{2}\sum_{i}(E(x_i,w)-y_i)^2 \end{align*} $$ We define this $L(w)$ as our Cost Function.

Now, we minimize $L$, differentiate w.r.t. $W$.
$$ \begin{align*} \nabla_w L(w) = \nabla_{w} \frac{1}{2} (XW-Y)^T (XW-Y) \\ = \frac{1}{2} \nabla_{w} \big((XW)^T(XW) - (XW)^T\bar{Y}\\ - \bar{Y}^T(XW) + \bar{Y}^T\bar{Y}\big)\\ = \frac{1}{2} \nabla_{w} \big(W^T(X^TX)W - W^TX^T\bar{Y}\\ - \bar{Y}^TXW + \bar{Y}^T\bar{Y}\big)\\ = \frac{1}{2} \big(W^T(X^TX)W - 2(X^T\bar{Y})^TW\big)\\ \end{align*} $$
Now, we know that $\nabla_{x}b^Tx = b$ and $\nabla_{x}x^TAx = 2Ax$ for a symmetric matrix $A$.
Also, to minimize this, we must have $\nabla_{w}L(w) = 0$. Hence, $$ \begin{align*} X^TXW = X^T\bar{Y}\\ \mathbf{\color{#2a51fa}W = (X^TX)^{-1}X^T\bar{Y}} \end{align*} $$
In this way, we also verified our closed form solutions from both the ways.
